A sleek, responsive, and centralized web dashboard providing quick access to essential Open Source Intelligence (OSINT) tools and resources. Featuring a visually immersive "Matrix" inspired interface.

## 🎯 Objective

This project was built to streamline the investigation workflow. By gathering highly effective open-source tools in one place, it provides researchers and cybersecurity enthusiasts with a fast, organized, and distraction-free environment.

## 🔥 Key Features

- **Intuitive UI & Iconography:** Clean menu navigation enhanced with professional FontAwesome icons for quick visual recognition.
- **Immersive Visuals:** A dynamic, custom-built Matrix digital rain effect running seamlessly in the background.
- **Responsive Design:** Fully adaptable layout that works perfectly on both desktop monitors and mobile devices.
- **Centralized Hub:** All critical OSINT resources aggregated into a single, easy-to-use control panel.
- **Secure Browsing:** All external tool links open in a new secure tab (`rel="noopener noreferrer"`).

## 🌐 Included Tools & Resources

The dashboard currently features direct links to:
- **Caipora** (Intelligence Tools)
- **Start.me OSINT Lists** (BR & Global)
- **NixIntel's Resource List**
- **OSINT4ALL**
- **MetaOSINT**
- **CTI (Cyber Threat Intelligence) Resources**

## 🚀 How to Run Locally

You don't need any complex server setup to run this dashboard on your machine.

1. **Clone or download** this repository to your local machine.
2. Navigate to the project folder.
3. Simply **double-click** the `index.html` file to open it in your default web browser.

## 💡 Technologies Used

- **HTML5:** Semantic structure and accessibility practices (`aria-labels`).
- **CSS3:** Flexbox-based layout, hover animations, and custom styling.
- **JavaScript (Vanilla):** Canvas API utilized for the real-time background animation.
